# 260702-2 更新內容

## 首頁刷新

首頁新增「刷新版本 / 重新載入」按鈕，適合 iOS 加到主畫面或捷徑無法手動刷新時使用。

按下後會：

1. 嘗試清除瀏覽器 CacheStorage。
2. 將網址加上新的 `v=260702-2-時間戳`。
3. 重新載入頁面，讓 GitHub Pages 取得新版 HTML / JS。

## 準備清單改版

準備清單改成完全自訂：

- 不再預設產生「護照、萬國轉接頭」等項目。
- 可以自行新增分類。
- 可以自行新增項目。
- 可以改名分類與項目。
- 可以刪除分類與項目。
- 可勾選確認。
- 可清空勾選或刪除全部清單。

## 注意

準備清單目前仍使用 `localStorage`，每個旅程分開保存於目前瀏覽器，不會同步到 Google Sheet。

因為這版使用新的 localStorage key：

`travel_prepare_checklist_v2::旅程名稱`

所以舊版 260702-1 的預設清單不會自動帶入。
