# 260702-4 準備清單操作體驗修正

## 主要修正

1. 勾選項目時不再整個重畫準備清單面板。
2. 新增項目時不再整個重畫面板，輸入框會保留焦點，手機鍵盤較不會自動收起。
3. 同步狀態改為局部更新，不再因「待同步 / 同步中 / 已同步」造成整個畫面閃動。
4. 新增項目後會清空同一個輸入框，方便連續輸入。
5. 保留 Google Sheet 同步與 localStorage 備援。

## 更新檔案

- index.html
- config.js
- prep-checklist.js

1. 打開準備清單。
