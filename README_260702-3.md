# 260702-3 更新內容

## 前端

- 準備清單介面移除說明文字，畫面更乾淨。
- 準備清單改為 Google Sheet 同步。
- 仍保留 localStorage 作為暫存備援：網路或 Apps Script 失敗時，不會立刻遺失本機資料。
- 版本號更新為 `260702-3`。

## Apps Script 必改

請打開 Apps Script，把 `apps_script_prep_checklist_patch_260702-3.gs` 內的 patch 加進去。

### 1. handleActionGET 裡加入

放在 `if (action === 'test')` 後面即可：

```javascript
if (action === 'prep_checklist_get') {
  const tripId = getParam(e, 'tripId', '');
  const tripName = getParam(e, 'tripName', '');
  return getPrepChecklist_(ss, tripId, tripName);
}
```

### 2. doPost 裡加入

放在 `if (action === 'test') { ... }` 後面、`if (!type)` 前面：

```javascript
if (action === 'prep_checklist_save') {
  return createJSONPOutput(savePrepChecklist_(ss, contents), '');
}
```

### 3. 把 PREP CHECKLIST FUNCTIONS 整段貼到 Code.gs 最下方

貼完後儲存，重新部署 Web App 新版本。
