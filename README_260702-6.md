# 260702-6 準備清單自動同步修正

## 更新內容

- 準備清單新增自動重試同步。
- 閒置、回到前景、網路恢復時會自動重新同步。
- 若暫時寫入失敗，不會永久停在「離線」，會定時重試。
- 沒有本機待同步資料時，會定期讀取 Google Sheet，讓不同裝置的準備清單能自動更新。
- 使用者正在輸入準備項目時，不會為了自動刷新而重畫清單，避免鍵盤被收起。

## 建議更新檔案

- index.html
- config.js
- prep-checklist.js
- README_260702-6.md
